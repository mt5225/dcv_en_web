    IT Architecture Visualization

    uinnova's MoMoDa is the leader in 3D visualization for CCTV surveillance. Using a WYSIWYG authoring tool, users can easily construct a 3D scene and position CCTV cameras. MoMoDa provides open API to be integrated with the security cameras system.
    Customers can visualize the 3D buildings and campuses with interactive real time video feeds. Use cases include public security and safety operations in subway, city core, school campus and airport.
